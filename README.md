# Cervical-Dysplasia

git clone https://github.com/fogg-lab/Cervical-Dysplasia.git

**Prerequisites**:  
- [Python 3.7 or higher](https://www.python.org/downloads/)
- urllib3 (you can use pip to install this package)
#### Usage:
```
python get_images.py
# then follow the prompts
```
